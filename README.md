### Prerequisites

## Getting Started Locally

1. Clone the repo to a local directory

```
git clone git@github.com:afs2015/PepperRPG.git
```

2. Create a new virtualvenv

```
cd PepperRPG
python3 -m venv venv
source venv/bin/activate
```
